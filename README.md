# Goose Goose Duck Cheat

鹅鸭杀辅助，大伙可以点个Star，功能在持续更新。

# 进度

1. 正在完善玩家被鹈鹕吃了的信息提示
2. 正在把Tkinter的界面用Pygame移植到游戏中
3. 优化程序占用

# 问题

1. 目前游戏开始需要按F2初始化数据
2. 程序Bug较多，容易读空内存报错
3. 鹅鸭杀每次更新内存位置都会改变，需要用CheatEngine抓取

# 功能

### 显示类

1. 查看其他玩家的位置，方框透视
2. 查看玩家死亡信息(被吃信息正在更新)

### 功能类

1. 穿墙
2. 调节移速
3. 移除迷雾
4. 无cd
5. 传送到别的玩家位置
